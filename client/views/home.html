Behind the Scenes

We compiled a large library containing 533 drug relevant targets with 179,807 active ligands, where each target was defined by its ligand set. For a given query molecule, its target profile is generated by similarity searching against the ligand sets assigned to each target, for which individual searches utilizing multiple reference structures are then fused into a single ranking list representing the potential target interaction profile of the query compound.

Reference: Xian Liu, Yuan Xu, Shanshan Li, Yulan Wang, Jianlong Peng, Cheng Luo, Xiaomin Luo, Mingyue Zheng, Kaixian Chen, Hualiang Jiang: In Silico target fishing: addressing a “Big Data” problem by ligand-based similarity rankings with data fusion. Journal of Cheminformatics, 2014, 6:33
